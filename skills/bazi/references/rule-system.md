# 古籍理法蒸馏索引

Use these source families as rule lenses, not as decorative citations.

## 子平真诠

Primary for 格局主导 charts:

- Month command as pattern entry.
- Useful god, auxiliary god, success, failure, rescue, and purity.
- Judge whether a named pattern is actually operative.
- A pattern name without support, rescue, or clarity is downgraded.

## 滴天髓阐微

Primary for 气势主导 charts:

- Five-element momentum, flow, blockage, clarity, turbidity.
- True/false strength, partiality, excess, insufficiency.
- Whether the chart should be balanced, followed, transformed, or opened by flow.
- If global momentum is decisive, ordinary pattern analysis must yield.

## 穷通宝鉴

Primary for 调候主导 charts:

- Month-based climate remedy.
- Cold/warm/dry/damp operating conditions.
- Heavenly stem priority by season.
- A useful god that worsens climate cannot be the first remedy.

## 渊海子平

Use for:

- Ten-god imagery.
- Classical pattern vocabulary.
- Traditional phrase bank and symbolic associations.
- Supporting but not overriding the selected leading mode.

## 三命通会

Use for:

- Broad pattern comparison.
- Luck and year corroboration.
- Case-like reasoning and old-school cross-checks.
- Secondary evidence in conflict sections.

## 千里命稿

Use for:

- Modern mingshu organization.
- Strength/weakness explanation.
- Six-god and six-kin presentation.
- Clear practical report structure.
